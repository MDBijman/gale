Template
c++ angle bracket syntax
type type_id<type_param_id ...> [requires constraints] = (body...)

constraint_id can be any constraint
type_param_id is used to refer to the type in the body

type Vec3<T> = (T x, T y, T z)

Constraints

std::is_number<T> requires T == i32
std::is_string_or_number<T> requires {
	T == i32 || T == str
}

can be used as

type Vec3<T> requires { T == i32 || T == f32 } = (T x, T y, T z)
type Vec3<T> requires std::is_number<T> = ()
